# WeUI - React

WeUI with Webpack, Babel, React & LESS.

## 使用说明 (编辑中）
[http://n7best.com/weui](http://n7best.com/weui)

## 开发

```
git clone https://github.com/n7best/react-weui
cd react-weui
npm install
npm start
```
然后在浏览器打开 `http://localhost:8080`。
